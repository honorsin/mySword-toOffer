## 2.善用位运算

1. **利用或操作** **`|`** **和空格将英文字符转换为小写**

```
('a' | ' ') = 'a'
('A' | ' ') = 'a'
```

1. **利用与操作** **`&`** **和下划线将英文字符转换为大写**

```
('b' & '_') = 'B'
('B' & '_') = 'B'
```

1. **利用异或操作** **`^`** **和空格进行英文字符大小写互换**

```
('d' ^ ' ') = 'D'
('D' ^ ' ') = 'd'
```

1. **判断两个数是否异号**

```
int x = -1, y = 2;
bool f = ((x ^ y) < 0); // true


int x = 3, y = 2;
bool f = ((x ^ y) < 0); // false
```
